| Feature | Description |
|---|---|
| PDF Upload | Upload any textbook PDF (up to 200MB) and extract text instantly via PyPDF2 |
| AI Study Notes | Concise, exam-ready notes generated by Gemini AI |
| MCQ Quiz Generator | Auto-generates 10 questions with a live timer, straight from your material |
| Doubt-Solving Chatbot | Answers strictly from your uploaded document — zero hallucination |
| Performance Analytics | Tracks quiz scores and progress over time |
| Secure Auth | Login/register system with session-protected dashboard |
| Cloud Database | MySQL-backed storage for documents and scores |

*Conceived, designed, built, and deployed by Harshali Alave — original author and lead developer.*
